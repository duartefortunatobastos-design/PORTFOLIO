1. Quem somos

Este portfólio é operado por Duarte Bastos (DB13 Website Desenvoloper / DB13 Server Solutions). É um website estático de apresentação pessoal, em português e inglês. Pode contactar-nos através do e-mail duartefortunatobastos@gmail.com.

2. Dados que o website trata diretamente

O código deste website não recolhe formulários, não utiliza ferramentas de analytics e não cria perfis de visitantes. Apenas guarda no seu navegador:

  • A preferência de consentimento de cookies (`portfolio-cookie-consent`), com o valor «aceitar» ou «fechar», para não voltar a mostrar o aviso de cookies.

A escolha de idioma (PT/EN) existe apenas durante a sessão de navegação e não é guardada automaticamente quando fecha o separador.

3. Alojamento e registos técnicos

O website é publicado através do GitHub Pages. Como qualquer serviço de alojamento, o GitHub pode processar dados técnicos padrão (por exemplo, endereço IP, pedido da página, data/hora e informação do navegador) nos seus próprios registos de infraestrutura. Não temos acesso a um painel de analytics nem enviamos notificações de visitas a partir deste portfólio.

4. Contacto e redes sociais

O website não inclui formulários de contacto. Se nos contactar por e-mail, WhatsApp, Instagram ou GitHub — através dos links disponíveis na secção de contactos — tratamos apenas as informações que nos enviar voluntariamente (por exemplo, nome, mensagem ou detalhes de um projeto).

5. Cookies e armazenamento local

Este website não define cookies HTTP próprios. Utiliza apenas armazenamento local estritamente necessário para memorizar a sua escolha no aviso de cookies. Consulte a Política de Cookies para mais detalhes.

6. Ligações para sites externos

O portfólio contém links para GitHub, Instagram, WhatsApp e projetos externos. Ao clicar nesses links, passa para plataformas de terceiros com políticas de privacidade próprias, sobre as quais não controlamos o tratamento de dados.

7. Os seus direitos

Ao abrigo da legislação aplicável de proteção de dados, pode solicitar o acesso, retificação, eliminação ou limitação do tratamento dos seus dados pessoais. Para efetuar um pedido, contacte-nos através do e-mail duartefortunatobastos@gmail.com.

8. Alterações a esta política

Esta política pode ser atualizada quando o website ou os serviços forem alterados. A versão mais recente estará sempre disponível nesta página.

Nota: Este texto descreve o funcionamento atual deste portfólio. Para validação jurídica formal, consulte um profissional de direito qualificado.
